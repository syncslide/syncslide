SyncSlide

New to SyncSlide? Read the presenter guide.

Even with the most accessible Powerpoint slides provided in advance, not being able to track what the presenter is showing on stage has been a pressing issue in academic settings, conferences, and business presentations for many with print disabilities.

About SyncSlide

SyncSlide is a presentation tool that incorporates web technologies to create simple, accessible, live slides that can be viewed from any device with a web browser. The goal is to create an alternative presentation software, such that assistive technology users (i.e., screen readers, specialized fonts, color inversion, etc.) can effortlessly follow along without the need to see the projecting screen. For presenters, SyncSlide provides a simple way to input their presentation content using basic text formatting (Markdown) in a semantical manner. They can navigate, edit the slides, and seamlessly update what's being shown in real-time. Since the application is built on web technologies, adding support for different types of content such as audience interaction tools, accessible graphics and math equations will no longer be a barrier.

With its focus on simplicity and versatility, SyncSlide aims to make presentations truly accessible to all.

FAQ

For Audiences

How does an audience member follow along?

Navigate to the same URL as the presenter (e.g. /username/presentation-id). You'll see a read-only view of the current slide that updates automatically as the presenter advances — no refreshing needed.

Do I need an account to join a presentation?

No. Audience members can view any presentation without logging in. An account is only required to create or edit presentations.

What devices and browsers are supported?

Any device with a modern web browser works — desktop, tablet, or phone. The audience view is compatible with screen readers and other assistive technologies out of the box.

Can I change the font, font size, or colors?

Yes — the audience view is plain HTML, so browser and OS accessibility settings apply directly. High contrast mode, custom fonts, and screen reader fonts all work out of the box.
