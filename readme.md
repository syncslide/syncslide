# SyncSlide

An accessible, Markdown-based presentation tool with real-time slide sync across all connected viewers.

Live instance: [syncslide.clippycat.ca](https://syncslide.clippycat.ca)

## Features

- Write slides in Markdown — each `## Heading` is one slide
- Audience sees the presenter's current slide in real time via WebSocket
- LaTeX support via KaTeX
- Server-side recording with synchronized slide playback
- Accessible-first: WCAG 2.2 AAA target, keyboard navigable, screen reader tested
- Share presentations with other users, or make them public

## Self-hosting

SyncSlide is a single Rust binary. See [CONTRIBUTING.md](CONTRIBUTING.md) for build, deploy, and configuration instructions.
